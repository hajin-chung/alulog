# alulog

my static markdown blog generator

## pages

1. /index.html : list of posts ordered in created date
2. /posts/\[id\].html
    - title
    - created at
    - updated at
    - tags
    - content
3. /tags : list of tags with number of posts in that tag
4. /tags/\[tag\].html : list of posts with tag ordered in created date
